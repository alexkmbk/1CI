Установка

Поскольку  1CI это самостоятельная конфигурация, то для работы с ней,  необходимо создать отдельную информационную базу. Для этого, необходимо запусить файл setup.exe.

Setup.exe скопирует файлы конфигурации в подкаталог OneCI каталога шаблонов 1С: Предприятие.

После этого,  необходимо создать информационную базу, выбрав в списке шаблон 1CI.

Первоначальная настройка в конфигураторе

Перед первым запуском в режиме предприятия, необходимо добавить нового пользователя с ролями Full access и Full administrator

Этот пользователь будет иметь права, необходимые для использования всех функциональных возможностей конфигурации.

Добавление хранилища в режиме предприятия

Откроем созданную информационную базу в режиме предприятия и добавим новое хранилище в справочник "Repositories".

Расшифровка некоторых полей, справочника Repositories:

Repository path - путь к каталогу, где находится хранилище;
Scheduled job user -пользователь, от имени которого будут выполняться регламентные задания с этим хранилищем.
Вкладка Repository users - список пользователей, которые будут иметь доступ к операциям с хранилищем.
