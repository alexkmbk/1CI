This list is used to store and view metadata object IDs. Data in this list are intended to maintain application operability and cannot be modified.

The list is filled in automatically based on configuration metadata, during the first configuration run (and each subsequent update), and updated automatically during each configuration version update (renamed, updated, and deleted metadata objects are processed).

The list displays, as a tree, all applied objects grouped by type. Object Synonym field is displayed by default. You can add Full name and ID fields to the list via All actions - Customize form command.
