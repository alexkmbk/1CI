User groups

This form is used to view and edit user group properties.

Users can be grouped by a variety of parameters, for example, by job title, or in accordance with corporate organizational structure. This facilitates access rights set up for multiple users with similar job duties.

Creating user groups

  • Enter group Description.
  • Select Group members:
    • When editing group members, keep in mind that they also may belong to the parent groups.
    • To add a large number of users to a group, use Pick command. It allows to select multiple users in user list and add them to a group with a single click.
    • The Add command only allows adding a single user at a time.
  • Included in group field can be used to make the new group subordinate to another group. Select the superior group in this field.
  • Comment field can be used to enter a detailed description of the group.

User group access rights

  • Click Access rights on group navigation panel. Click Include in group and select an access group from list.
  • If roles are assigned directly, you can assign roles both to single users and to user groups.
