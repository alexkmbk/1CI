This list is used to allow entering OS logon user name for application logon purposes.

This allows users to log on to the application using their OS logon name and password.

The list consists of two sections:

  • Domains section (on the left) displays the list of domains visible from this computer.
  • Users section (on the right) displays the list of users registered in the selected domain.

Select user

  • Select a domain.
  • Select a user from this domain and click Select, or double-click the user.
