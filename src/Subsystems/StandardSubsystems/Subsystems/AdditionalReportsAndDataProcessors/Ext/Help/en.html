This configuration is used to link additional (external) reports and data processors to infobases and use them, in 1C:Enterprise mode. This allows extending the application's functionality without making any changes to the application.

You can use the additional reports and data processors to:

  • In object forms and object lists:
    • generate reports related to the object;
    • create additional print forms for the object;
    • create and fill in the subordinate (child) objects based on the parent objects;
    • fill in the attributes for tabular sections of the object.
  • In command interface:
    • process any infobase data;
    • generate any reports.
