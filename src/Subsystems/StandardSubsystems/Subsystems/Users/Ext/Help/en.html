The system is intended for managing users and external users. It synchronizes them with the infobase user list (to view the list, run the application in Designer mode). This provides you with access to user details, as well as with the option to reference to users from other infobase objects.

 

Important: we recommend that you manage infobase user list and set up user access in 1C:Enterprise mode only. In Designer mode, only users with full access (system administrators) can be created. When a user with full access starts the infobase in 1C:Enterprise mode, corresponding items will be automatically generated in the Users catalog.
