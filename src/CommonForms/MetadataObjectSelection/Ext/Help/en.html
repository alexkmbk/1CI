This list is used to select applied objects.

It is called from other applied objects via Select command or corresponding hyperlink.

The list displays, as a tree, all applied objects grouped by type.

Selecting applied objects

To select one or more applied objects, click the corresponding check boxes.

  • To select all applied objects of a specific type, click the corresponding group-level check box (example: click Catalogs check box to select all catalogs).
  • To select all applied objects in the list, click the root-level check box.

Click Select.
