This form is used to enter the email account passwords. It is displayed whenever you attempt to send an email message from an account with no password specified.

Entering passwords and sending messages

  • To send your message, enter Password and Confirmation;
  • Click  Continue to confirm the password and send the message.
  • Click Cancel to clear the password and cancel sending the message.

Storing passwords

  • Select the Save password check box if you want to store the password with the infobase system settings. When creating a system email account, we recommend that you select this check box so that email messages from this account can be sent without human intervention. However, you are free to choose whether to store any account passwords in the infobase or not.
