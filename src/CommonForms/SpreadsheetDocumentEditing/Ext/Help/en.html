This form is used to edit spreadsheet documents.

To start editing a spreadsheet document or template, click  Open template for edit located in command bar of the print form or spreadsheet document.

Important: when editing print form templates, keep in mind that any changes to area names or template parameters can make the print form unusable. The print form can be reverted to default settings via Print form templates list.

Editing

  • Standard formatting buttons are available for editing purposes: changing  font size and typeface,  background color and font color, alignment, borders, and more.
  • You can also  Merge cells and unmerge cells via All actions command - Unmerge cells. 
  • You can also  Insert comment.
  • When finished with your edits, click Save and close.

Switching between view and edit modes

To switch between modes, click Edit. In view mode, the template edit buttons are disabled.
