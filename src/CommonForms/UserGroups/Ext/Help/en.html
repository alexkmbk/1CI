This list is used to add or remove users (external users) from groups.

To view the list of a user's groups, execute Groups command available on the user (external user) navigation panel.

Adding users to groups

  • Select check box corresponding to the group you want to add the user to
  • Click Save and close.

Removing a user from all groups

This function can be helpful when a user was added to multiple groups by mistake. You can quickly remove a user (external user) from all groups and add them again as needed.

  • Click Exclude from all groups. Click Write if necessary.
