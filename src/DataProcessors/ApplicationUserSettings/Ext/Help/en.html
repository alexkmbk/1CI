This form is used for user settings management.

User settings management commands are located in the Administration - User and rights settings section.

Only administrator is allowed to manage user settings.

Available commands

  • Сopy settings - this command copies user settings between users. It can be helpful when you need to configure multiple new users in a group already containing users.
  • Clear settings - this command clears settings for all or selected users. In certain cases it may be necessary to clear user settings - for example, when invalid settings prevent normal application use.
  • User settings - this command opens the user settings management list.
