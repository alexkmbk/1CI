This form is used to copy user settings between application users. You can only copy settings between users belonging to the same list - Users or External users

It can be helpful when you need to configure multiple new users in a group already containing users.

To copy user settings, execute command in the Administration - User and rights settings section.

Copy application user settings

  • In the Copy settings from user field, select a user (external user) whose settings will be copied to other users (external users):
    •  click  Select and specify the Users (External users) list where you want to copy user settings. 
    • in the Select user (Select external user) list, select a user (external user) and click Select, or double-click the user
  • In the Copy to field, select users to whom the settings will be copied. You can set the radio button to:
    • Selected users - this requires you to specify users to whom the settings will be copied. Click the Select hyperlink.
      • User selection to copy settings window will open. Click check boxes to select users (external users).
      • Once the selection is complete, the hyperlink will change to display the name of the single selected user or total number of the selected users.
    • All users - this does not require any additional actions from you. You may proceed to the next step.
  • In the Copy what field, select settings to be copied between users:
    • All settings - this does not require any additional actions from you. You are ready to start copying the settings.
    • Selected settings - this requires you to specify settings from list. Click the Select hyperlink.
      • User settings selection window will open. Click check boxes to select settings to be copied.
      • Once the selection is complete, the hyperlink will change to display the name of the single selected user setting, or total number of the selected settings.
  • Click Copy and close to copy the settings with selected parameters and go back to the application, or click Copy to copy the settings and continue working with this form.

Restrictions

Some of the user settings cannot be copied even when All settings is selected.

  • When you copy application appearance settings to any users, these users must terminate their application sessions. Otherwise, the settings may not be applied correctly
  • Before copying report option settings, make sure that recipient users have sufficient rights for working with these reports.
  • Personalized report options cannot be copied between users.

If some of the settings were not copied, the application will display a warning message. Click Show report to review the list of settings that were not copied.
