This form is used to clear user settings.

In certain cases it may be necessary to clear user settings - for example, when invalid settings prevent normal application use. Any settings, including personalized report options, can be cleared.

To clear user settings, use the Administration - User and rights settings command.

Clear application user settings

  • Use the Clear settings radio button to specify which users should have their settings cleared:
    • Selected users - click the Select hyperlink;
      • Select the Users (External users) list;
      • Select users from the User selection list via check boxes. Click Select;
      • Once the selection is complete, the hyperlink will change to display user name (if a single user was selected), or the number of selected users (if multiple users were selected).
    • All users - in this case, only the All settings option will be available.
  • Use the Clear what radio button to specify which settings should be cleared:
    • All settings - when multiple users or all users are selected, only the All settings option will be available for each user.
    • Selected settings - when a single user is selected, you can specify which settings should be cleared for this user. 
      • Click the Select hyperlink.
      • The User settings selection window will open. Select settings to be cleared via check boxes.
      • Once the selection is complete, the hyperlink will change to display the name of a single selected user setting, or total number of the selected settings.
  • Click Clear and close to clear the settings with selected parameters and go back to the application, or click Clear to clear the settings and continue working with this form.
