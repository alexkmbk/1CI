This list is used for user settings management.

Here you can view, clear, or copy report user settings, application appearance settings, and other settings.

To access the settings list, use the User settings command in the Administration - User and rights settings section.

  • In this case, the list will display the User field. The User field is mandatory. It is filled with the Administrator value by default. If any external users are registered in the application, you can manage their settings as well.

To access settings list for any user (external user), you can also use the Settings command on the navigation panel.

  • In this case, the User field will not be displayed.

To locate a specific setting quickly, type full or partial name of the setting into the Find field.

The list consists of three tabs:

  • Appearance - settings controlling the appearance of application desktop, forms (application lists, etc), and command interface.
  • Report settings - user-defined report settings, grouped by report option name.
  • Other settings - user's personal settings, spreadsheet document printing settings, settings for quick access to additional reports and data processors, the Favorites section settings, and other settings.

Сopy settings

Select user in the User field. The following commands are available:

  • Copy to others - use this command to copy the selected settings of this user to one or several application users.
    • Select a user;
    • Select settings to be copied by clicking, Ctrl-clicking and Shift-clicking;
    • Select the required users from the User selection list via check boxes. Click Select. The settings will be copied.
  • All actions - Copy settings group to others - identical to the above. The following settings groups are available in the command menu:
    • Report settings;
    • Appearance settings;
    • Report and appearance settings;
    • Report, appearance, and other settings - all user settings will be copied.
  • Copy from - use this command to copy user settings from another application user to the selected user:
    • select user whose settings you want to copy;
    • select settings you want to copy. This is identical to the Сopy settings command in the Administration - User and rights settings section. 

Restrictions

Some of the user settings cannot be copied even when All settings is selected.

  • When you copy application appearance settings to any users, these users must terminate their application sessions. Otherwise, the settings may not be applied correctly. To view the list of current user sessions, use the Active users... command in the User selection list.
  • Before copying report option settings, make sure that recipient users have sufficient rights for working with these reports.
  • Personalized report options cannot be copied between users.

If some of the settings were not copied, the application will display a warning message. Click Show report to review the list of settings that were not copied.

Clearing settings for selected user

Select user in the User field. The following commands are available:

  • Clear - this command clears selected settings of the selected user (select settings to be cleared by clicking, Ctrl-clicking and Shift-clicking).
  • Clear all - select user settings groups:
    • All settings;
    • Report and appearance settings.

Clearing selected settings for selected users

  • Select settings to be cleared by clicking, Ctrl-clicking and Shift-clicking;
  • Click All actions - Clear all - Selected user settings;
  • Click Yes to confirm;
  • Select the required users from the User selection list via check boxes.
  • Click Select to clear settings.

Clearing all settings for all users

  • Click All actions - Clear all - All settings of all users;
  • Click Clear all to confirm.

Viewing current user settings

You can only view settings of the current application user. You can also preview report and application appearance with the selected settings apllied.

  • Select a setting;
  • Click All actions - Open.
