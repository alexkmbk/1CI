This form is used to view properties of application events registered in event log.

It displays full information (more detailed than event log data) on each application event:

  • User - full user name
  • Date, time - exact event time;
  • Application - Thin client, etc; 
  • Computer - network ID of the computer;
  • Data field group:
    • Metadata - applied object type and description: list, document type, etc;
    • Data - applied object indication: name of specific list item, document, etc;

    • Data presentation - applied object indication.

  • Event field group:
    • Event - event type;
    • Comment - brief description of the event;
  • Session field group - full information on the session during which the event occurred:
    • Number - session number; 
    • Working server;
    • Port;
    • Sync port.
  • Transaction field group:
    • ID;
    • Transaction completion status - Committed, etc.

You can choose to display only some of the columns in the event log, hiding any information that is not necessary for your purposes.
