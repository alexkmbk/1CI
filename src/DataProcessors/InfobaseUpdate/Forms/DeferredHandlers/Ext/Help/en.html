This list is used to view the additional data processors executed after infobase version update, in deferred mode in background.

To open the list, execute the Update results and additional data processors command located in the Administration - Support and maintenance section. The Application update results window will open. In this window, click the related hyperlink.

The list includes:

  • Data processor - name and detailed description for the additional data processor
  • Execution status - brief information on how the additional data processor was executed.

Select a data processor to view details of its execution at the bottom of the list.

Filters

  • Status - select execution status in this field (All data processors, Error, Executing, Completed);
  • Find - type in a full or partial word into this field to find it.

Executing additional data processors

Click Run to start executing the additional data processors. This button is only available until any additional data processors are started, or while any data processors are waiting for execution.

Viewing event log

To view the additional data processor events in the event log, click the Technical details on data processing results hyperlink.
