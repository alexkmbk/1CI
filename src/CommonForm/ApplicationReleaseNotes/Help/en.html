The update details include the list of changes for the latest application version, as well as change history for earlier versions.

This information is automatically displayed once after the application version is updated. To view the update details at any time, execute the Application update details command located on the Information panel on desktop.

Available commands

  • The application update details can be  saved to file or  printed by executing the corresponding commands. 
  • For easier navigation, you can collapse or  Expand all groups (headers).
  • Text search feature is available if you need to find specific update information quickly.

Infobase version update progress information

  • If data processors are still running after the application version is updated, a warning will be displayed for all users: Additional data processors in progress. For this duration, the application functionality will be temporarily limited, as transition to the new version is not yet complete.

Administrative features

  • Administrator can review the list of optional instructions recommended for execution after the mandatory infobase update procedure completes. To execute the instructions, click the hyperlinks located in the new version description.
  • Click the Additional data processors in progress hyperlink to monitor the additional data processors running.
  • To view the infobase update events in the event log, click the Technical details on update results hyperlink.
