Click Reconnect to reconnect the master node before resuming execution. This operation must be performed before loading master node configuration to a subordinate node during distributed infobase recovery, if the master node was previously disconnected (earlier configuration versions cannot be loaded).

Click Disconnect to irrevocably disconnect the master node before resuming execution. This operation must be performed in the following cases:
- when creating a master node from a copy of subordinate node;
- when creating an independent infobase from a subordinate node (or a copy of subordinate node).
