If 1C:Enterprise server cluster uses infobase security profiles that limit access to external (relative to 1C:Enterprise server cluster) resources, security profile permissions must be modified for the application to work correctly.

The list displays external resources permitted to be used.

To confirm, click Apply changes.

Moreover, you need to ensure that: 

  • access to the external resources is not blocked by any software installed on the server (such as antiviruses, traffic control utilities, etc);
  • all necessary rights were provided to operating system users on behalf of which the servers working in 1C:Enterprise cluster are running.
