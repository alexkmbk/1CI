This form is used to select one or several users whose settings you want to copy or clear.

The list consists of two sections: the left section displays groups, and the right section lists users (external users) in each of these groups. All users (All external users) group is selected by default.

To display all users in the nested hierarchical groups, select the Show nested group users check box at the bottom of the list.

User selection

  • Select users by clicking corresponding check boxes. You can also:
    • Click to Mark all users;
    • Click  to Clear marks for all users;
    • Select users by clicking, Ctrl-clicking and Shift-clicking, then click All actions - Mark selected items.
    • Select the marked users and click All actions - Clear marks of selected items.
  • To copy or clear settings, click Select.

Viewing active users

When you copy application appearance settings to any users, these users must terminate their application sessions. Otherwise, the settings may not be applied correctly.

  • To view the list of current user sessions, use Active users command.
