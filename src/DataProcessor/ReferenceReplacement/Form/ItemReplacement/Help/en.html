This form is used to replace one or more items with a specific item.

This functionality is most useful when deleting items, as you need to delete all references to an item before the item itself can be deleted.

The Replace everywhere where found command is available in the Deletion of marked objects window.

This way, you can automatically replace all references to the items you want to delete with references to another item.  All selected items will be marked for deletion, or deleted directly if you have sufficient administrative rights.

Item replacement

All items of the corresponding list are displayed, including ones marked for deletion.

  • Click  to view the selected item.
  • Filter by period is available for documents. If the list contains an excessive amount of documents, you may want to use this filter. Click  and Set view date interval.
  • Select an item which you want to replace the specified items.
  • Infobar at the bottom of the list displays attributes of both the items to be replaced and the item used for replacement. If you have administrative rights, you will be provided with an additional option. You can follow a hyperlink and specify whether the replaced items should be:
    • marked for deletion;
    • deleted permanently.
  • Click Replace to replace all references.

If any errors are encountered during the replacement procedure, a list of items that failed to be processed will be displayed, complete with description of failure causes. In this case, you will have the following options:

  • Open item for viewing by clicking  or double-clicking the item;
  • Retry item replacement, by clicking Retry replacement;
  • Click Back and select another item for replacement;
  • Click Close and go back to the application.
