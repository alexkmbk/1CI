This list is used for management of email accounts used to send or receive email messages.

To open the list, execute the corresponding command located in the Administration - Organizer section. 

The list includes:

  • Name - name of the account;
  • User name - the name of message sender. It can differ from the user name registered in the application.
  • Email address - account address in the user@mailserver format. 

If you want to send and receive email messages automatically, you need to specify the password for this account. Moreover, you need to set up the system email account in order to ensure the application works correctly.

System email account

A predefined system email account is available in the application. It is used by default to send a variety of notification messages related to the application. For example, it can be used to notify the administrator that the application update procedure has completed successfully. Before you can start using this fucntionality, you need to set up the system email account.

  • To set up the system email account, execute the corresponding command located in the Setup and administration - Organizer section. You can also Change it in the list.

Creating email accounts

Before saving the email account settings, they will be validated (a test email message will be sent). If validation fails the account settings will not be saved.

  • Click Create and fill in the required fields:
    • Specify account Email address in the user@mailserver format;
    • Enter account Password. When creating a system email account, we recommend that you select the Remember password check box so that email messages from this account can be sent without human intervention. However, you are free to choose whether to store any account passwords in the infobase or not;
    • Specify Account usage options by selecting check boxes:
    • Sending messages;
    • Receiving messages;
  • Specify Sender name (your name, or name of your company);
  • Use the radio button to select account creation method:
    • Detect connection parameters automatically (recommended);
      • Click Create. Account will be set up and validated automatically; just follow the instructions. If automated setup fails, you can proceed with the manual parameter input (see below for details). 
    • Set up connection parameters manually;

  • Click Next to fill in other parameters. For many mail servers, recommended settings will be filled in automatically. You will only need to validate and save the account settings.

    Setting up email accounts

    • Click  Change and modify the settings accordingly.

    Validating email accounts

    • Select an email account from list and click All actions - Validate account to validate the email account parameters. This will send a standard email message in order to check that connection to incoming mail server can be established (provided that the email account is used to receive email).
    • If password is not specified in the account settings, you will be prompted to enter it. Enter password if necessary.
